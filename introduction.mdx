Llama Stack is a drop-in replacement for the OpenAI API that you can run anywhere — your laptop, your datacenter, or the cloud. Use any OpenAI-compatible client or agentic framework. Swap between Llama, GPT, Gemini, Mistral, or any model without changing your application code.

```python
from openai import OpenAI

client = OpenAI(base_url="http://localhost:8321/v1", api_key="fake")
response = client.chat.completions.create(
    model="llama3.2:3b",
    messages=[{"role": "user", "content": "Hello"}],
)
print(response.choices[0].message.content)
```

## Get started in 3 steps

<Steps>
  <Step title="Install and start the server">
    ```bash
    uvx --from 'llama-stack[starter]' llama stack run starter
    ```
    The server starts at `http://localhost:8321`.
  </Step>
  <Step title="Connect with any OpenAI client">
    ```python
    from openai import OpenAI

    client = OpenAI(base_url="http://localhost:8321/v1", api_key="fake")
    response = client.responses.create(
        model="llama3.2:3b",
        input="What is Llama Stack?",
    )
    print(response.output_text)
    ```
  </Step>
  <Step title="Swap providers as you scale">
    The same API works whether your backend is Ollama, vLLM, OpenAI, AWS Bedrock, or dozens of other services. [See all providers](/providers/overview).
  </Step>
</Steps>

## Client SDKs

Connect from any language using the official SDKs or any OpenAI-compatible library.

| Language | SDK | Install |
|----------|-----|---------|
| Python | [llama-stack-client-python](https://github.com/meta-llama/llama-stack-client-python) | `pip install llama-stack-client` |
| TypeScript | [llama-stack-client-typescript](https://github.com/meta-llama/llama-stack-client-typescript) | `npm install llama-stack-client` |

<Tip>
  Since Llama Stack is OpenAI-compatible, you can also use the standard `openai` Python or Node.js SDK directly — just point `base_url` at your Llama Stack server.
</Tip>
